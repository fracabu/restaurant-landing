<a name="english"></a>
## :gb: English

### Overview

A modern restaurant landing page built with Vite, TypeScript, and Tailwind CSS. Responsive design optimized for showcasing menus, locations, and restaurant information.

### Features

- **Menu Showcase** - Display dishes and prices
- **Location Info** - Maps and directions
- **Responsive Design** - All devices supported
- **Reservations** - Booking integration ready
- **Photo Gallery** - Showcase ambiance

### Quick Start

```bash
git clone https://github.com/fracabu/restaurant-landing.git
cd restaurant-landing
npm install
npm run dev
```

Access at `http://localhost:5173`
